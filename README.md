### Installation

1. ```git clone https://github.com/codingtho/tho-zsh-theme.git```
2. ```cp tho-zsh-theme/tho.zsh-theme ~/.oh-my-zsh/themes/```
3. Open your `.zshrc` file and configure the zsh theme
```ZSH-THEME="tho"```
4. Save changes
```source ~/.zshrc```
